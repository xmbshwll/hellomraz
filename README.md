# Astro Blog

A [Опусти Оружие, Мразь](https://t.me/hellomraz) web version built with [Astro](https://astro.build), featuring a beautiful design system from [williamcachamwri/astro-blog](https://github.com/williamcachamwri/astro-blog), and hosted on GitHub Pages.

[Опусти Оружие, Мразь](https://t.me/hellomraz) is a DIY music blog with completely subjective on author taste fast frestyle reviews on music releases and other related stuff, primarily focused on punk / noise and other underground heavy music. It is our job to keep punk rock elite.
